# Excel и данные операций

## Файл

- Путь: `exports/Fuel_Report_Master.xlsx` (создаётся при первой записи).
- Листы: `Заправки_по_картам`, `Заправки_личные_средства`, `Спорные заправки`, `Справочники`.

Реализация: `src/app/excel_export.py`.

## Личные средства

Для `source=personal_receipt` в строку подставляются поля из `ocr_data` (топливо, литры, сумма, АЗС и т.д.), колонка «Данные OCR» заполняется из `raw_text` / `raw_text_debug`.

Колонка «Кто первоначально получил запрос» для таких операций берётся из `presumed_user_id` (кто прислал фото).

## Справочник автомобилей

Проверка госномера при сценарии личных средств и при вводе номера по карте опирается на таблицу **`cars`** и нормализацию номера (`src/app/plate_util.py`). Автомобиль должен существовать в БД; произвольный ввод без записи в `cars` не завершает сценарий.

## Пользователи

ФИО заправившегося ищется среди активных записей **`users`** (`active=True`). Импорт и карты не описываются здесь — см. код импорта и админ-хендлеры.
