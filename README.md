# fuel-tracker-bot

# 1.Руководство по запуску

- pip install - r requirements.txt
- создать БД в postgreSQL и вставить свою ссылку в .env (таблицы создаются автоматически при первом запуске)
- установить Tesseract на своё устройство (https://github.com/UB-Mannheim/tesseract/wiki)
- запуск через run_bot.py
- запуск тестовой операции заправки через run_mock_import.py (в конце файла run_mock_import.py необходимо указать свой телеграм id)
